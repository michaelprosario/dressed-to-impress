Here are some helpful YouTube videos to walk you through using Google Cloud Storage (GCS) buckets, ranging from quick overviews to step-by-step guides:

* **[Create and use a Cloud Storage bucket](http://www.youtube.com/watch?v=F4XFrHhhLow)** (Google Cloud Tech)
A super quick, under-a-minute visual walkthrough from Google showing the absolute basics of spinning up a bucket and putting it to use.
* **[GCP Cloud Storage Buckets: Complete Beginner’s Guide!](http://www.youtube.com/watch?v=2DuF8-UIoU4)** (theAI Dev)
A concise 2-minute primer perfect if you want a fast breakdown of what buckets are and how to start navigating them as a beginner.
* **[Bucket options in Cloud Storage](http://www.youtube.com/watch?v=8DMOJ6Lgm7s)** (Google Cloud Tech)
This 3-minute video is great for understanding the *settings* you choose when creating a bucket, like storage classes (Standard vs. Coldline) and location options.
* **[How to create Storage Bucket in GCP](http://www.youtube.com/watch?v=Px1oXqR1bFA)** (BestTechLearn)
A longer, more detailed 10-minute tutorial that takes a slower, step-by-step approach to creating and configuring your storage inside the console.
* **[How to Set Buckets and Files Public In Google Cloud Storage](http://www.youtube.com/watch?v=3V8aDWRreFU)** (Jie Jenn)
If your specific goal is to host files or assets that need to be accessed via a public URL, this 3-minute video covers permissions and access control lists (ACLs).
